This project is a responsive, space-themed personal portfolio developed using 
<strong>React</strong>, <strong>Tailwind CSS</strong>, and <strong>React Router DOM</strong>. 
It includes sections for my skills, projects, background, and contact information, 
with a modern and minimal user interface optimized for both desktop and mobile.
</p>

<h2>Features</h2>
<ul>
  <li>Developed with React using a modular, component-based structure</li>
  <li>Styled with Tailwind CSS</li>
  <li>Theme support including Dark Mode</li>
  <li>Fully responsive design for mobile and tablet</li>
  <li>Routing implemented using React Router DOM</li>
  <li>Custom 404 Not Found page</li>
  <li>Sections: About, Skills, Projects, Contact</li>
</ul>

<h2>Tech Stack</h2>
<ul>
  <li>React</li>
  <li>Tailwind CSS</li>
  <li>JavaScript (ES6+)</li>
  <li>React Router DOM</li>
  <li>Vite</li>
</ul>
